# bayesfoRest: Forest Plots for Bayesian Meta Analyses in R

bayesfoRest creates forest plots for Bayesian meta-analyses in data that
has been prepared using the metafor and brms packages. In addition to
forest plots, posterior probability plots are generated for the overall
effect and for the heterogeneity parameter.
